# カテゴリ別プレースホルダー対応 - 完了報告

## 変更概要

FlagNFT管理ページの基本情報入力ステップで、選択されたカテゴリに応じて適切なプレースホルダーテキストを表示するように改善しました。

## 変更ファイル

- ✅ `src/admin/components/FlagNFTManagementPage.tsx`

## 実装内容

### 1. NFT名のプレースホルダー

選択されたカテゴリに応じて、以下のようにプレースホルダーが動的に変更されます:

| カテゴリ | プレースホルダー例 |
|---------|------------------|
| 💳 BENEFIT（特典NFT） | 例: 10%割引特典NFT |
| 👤 MEMBERSHIP（会員証NFT） | 例: ゴールド会員証NFT |
| 🏆 ACHIEVEMENT（実績バッジNFT） | 例: 100回投げ銭達成バッジ |
| 🎪 CAMPAIGN（キャンペーンNFT） | 例: 夏季限定スタンプラリー |
| 🗝️ ACCESS_PASS（アクセス権NFT） | 例: VIPラウンジ入場パス |
| 🎴 COLLECTIBLE（コレクティブルNFT） | 例: 限定アートコレクション #1 |

### 2. 説明のプレースホルダー

カテゴリごとに具体的な説明文の例を提供:

#### 💳 BENEFIT（特典NFT）
```
この特典NFTで受けられる割引や特典内容を詳しく説明してください
（例: カフェメニュー全品10%オフ）
```

#### 👤 MEMBERSHIP（会員証NFT）
```
会員証の特典内容や利用できるサービスを説明してください
（例: VIPエリアアクセス、限定イベント招待）
```

#### 🏆 ACHIEVEMENT（実績バッジNFT）
```
この実績バッジの達成条件と獲得時の特典を説明してください
（例: 累計100回投げ銭で獲得、特別称号付与）
```

#### 🎪 CAMPAIGN（キャンペーンNFT）
```
キャンペーンの内容、参加方法、達成報酬を説明してください
（例: 店舗5箇所を巡るスタンプラリー、完走で限定グッズ）
```

#### 🗝️ ACCESS_PASS（アクセス権NFT）
```
アクセス権の利用可能範囲と有効期限を説明してください
（例: VIPラウンジ入場権、イベント当日のみ有効）
```

#### 🎴 COLLECTIBLE（コレクティブルNFT）
```
コレクションの背景やアーティスト情報を説明してください
（例: 限定100枚のデジタルアート、著名イラストレーター作）
```

## UX改善効果

### Before
- 全カテゴリで同じプレースホルダー「例: 10%割引特典NFT」
- ユーザーは各カテゴリでどのような名前・説明を入力すべきか迷う

### After
- ✅ カテゴリごとに適切な例示
- ✅ 具体的な使用例の提示
- ✅ 入力内容のガイダンス強化
- ✅ UX向上とエラー入力の削減

## 動作確認

### 確認手順

1. http://localhost:5173 にアクセス
2. テナント管理者でログイン
3. ダッシュボード → フラグNFT管理
4. 「新規作成」をクリック
5. 各カテゴリを選択して基本情報ステップへ進む
6. NFT名と説明欄のプレースホルダーが各カテゴリに応じて変わることを確認

### テスト例

**特典NFTを選択した場合:**
- NFT名プレースホルダー: `例: 10%割引特典NFT`
- 説明プレースホルダー: `この特典NFTで受けられる割引や特典内容を詳しく説明してください（例: カフェメニュー全品10%オフ）`

**会員証NFTを選択した場合:**
- NFT名プレースホルダー: `例: ゴールド会員証NFT`
- 説明プレースホルダー: `会員証の特典内容や利用できるサービスを説明してください（例: VIPエリアアクセス、限定イベント招待）`
