# プランヘッダー画像

このディレクトリには、各テナントプラン専用のヘッダー画像を配置します。

## 必要な画像ファイル

以下のファイル名で画像を配置してください：

- `studio.png` - STUDIOプラン用ヘッダー画像
- `studio-pro.png` - STUDIO PROプラン用ヘッダー画像
- `studio-pro-max.png` - STUDIO PRO MAXプラン用ヘッダー画像
- `default.png` - デフォルト/フォールバック用ヘッダー画像

## 推奨サイズ

- **横幅**: 800px以上
- **縦幅**: 160px（デスクトップ）/ 120px（モバイル）
- **アスペクト比**: 5:1 を推奨
- **フォーマット**: PNG または JPEG

## 使用箇所

これらの画像は以下で使用されます：
- マイページの承認済みテナント向けプランカード（`TenantPlanCard.tsx`）
- プラン別にヘッダー画像が自動的に切り替わります

## フォールバック動作

画像の読み込みに失敗した場合、プランアイコン（絵文字）が代わりに表示されます。
